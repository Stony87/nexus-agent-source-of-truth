# Agent Registry

This directory contains the official charter for each agent in the Nexus control-plane.

An agent is not part of the system until it has a file here, committed and approved.

## Registered Agents

| Agent | File | Status | Role |
|---|---|---|---|
| Nexus | [nexus.md](nexus.md) | Active | Control-plane orchestrator |

## How to Register a New Agent

1. Draft a charter using the structure in any existing agent file.
2. Open the draft for Anthony's review.
3. Once approved, commit the file here and update this index.
4. Record an ADR in [docs/decisions/](../decisions/) documenting the decision to add the agent.
